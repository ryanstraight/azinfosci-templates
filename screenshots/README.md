# Template Screenshots

This directory contains preview images for each template.

## Adding Screenshots

To generate screenshots:

1. Render each template with sample content
2. Take PNG screenshots at reasonable resolution
3. Name files to match template names

## Recommended Files

- `presentation.png` - Title slide or representative slide
- `poster.png` - Full poster at reduced size
- `document.png` - First page of rendered document
- `letter.png` - Rendered letter
- `memo.png` - Rendered memo
- `assignment.png` - Assignment handout
- `agenda.png` - Meeting agenda
- `one-pager.png` - Fact sheet
- `lab-notebook.png` - Lab notebook page
- `report.png` - Report title page
- `working-paper.png` - Working paper first page

## Recommended Dimensions

- Width: 800-1200px
- Format: PNG
- Keep file sizes reasonable (<500KB each)
