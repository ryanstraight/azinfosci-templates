# University of Arizona Logos

Due to trademark restrictions, official University of Arizona logos cannot be distributed with this template collection.

## Obtaining Logos

Download official logos from the UA Brand Resources website:

**<https://brand.arizona.edu/brand-assets>**

You will need UA NetID authentication to access the download portal.

## Required Files

Place the following files in this directory for full template functionality:

| Filename | Usage |
|----------|-------|
| `infosci-logo.png` | Primary College of Information Science logo (horizontal) |
| `infosci-logo-white.png` | White/reverse version for dark backgrounds |
| `ua-logo.png` | Main University of Arizona wordmark |

## Logo Guidelines

When using UA logos, follow the official brand guidelines:

- Maintain clear space around logos
- Do not alter colors or proportions
- Use appropriate versions for light/dark backgrounds

See the [UA Brand Guidelines](https://brand.arizona.edu/) for complete usage requirements.
