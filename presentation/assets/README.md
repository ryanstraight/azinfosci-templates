# Presentation Template Assets

Place your logo files here:

| Filename | Description |
|----------|-------------|
| `infosci-logo.png` | College of Information Science logo (dark version) |
| `infosci-logo-reverse.png` | White/reversed logo for dark slide backgrounds |

## Obtaining the Logos

See the main package README or `inst/assets/logos/README.md` for instructions on obtaining official UA College of Information Science logos.

- Use `UA_College_of_Information_Science_PRIMARY.png` → rename to `infosci-logo.png`
- Use `UA_College_of_Information_Science_PRIMARY_REVERSE.png` → rename to `infosci-logo-reverse.png`
